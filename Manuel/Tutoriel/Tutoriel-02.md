# Eventer tutoriel (suite)

Nous allons donc  créer notre premier évènemencier pour notre nouveau projet. Dans une bonne utilisation, on commence par faire les actes. Imaginons que vous ayez trois actes.

- votre projet est sélectionné dans la fenêtre des projets

- vous jouez la touche →

  => vous « entrez » dans le projet et un premier event est prêt

- vous écrivez par exemple « Acte I »

- vous validez avec ↩︎

  => Bravo ! vous venez de créer votre premier event !

- pressez « n » pour créer un deuxième event et nommez-le « Acte II »

- créez l’acte III de la même manière.

Vous venez de créer le premier évènemencier avec *Eventer* ! 🥳

Il devrait ressembler à ça : 

~~~
Acte I
Acte II
Acte III
~~~

Maintenant, vous pouvez vous amuser avec la combinaison ⌘↓/⌘↑ pour déplacer chaque event. Ça marchera avec tous les panneaux, toutes les listes que vous pourrez utiliser dans *Eventer*.

N’oubliez pas de revenir au bon ordre avant de poursuivre ! 🤪
