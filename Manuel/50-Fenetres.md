## Les fenêtres

Pour une navigation encore plus pratique, on peut diviser la fenêtre principale pour obtenir deux évènemenciers.

Voici les raccourcis pratiques pour le réaliser :

| Raccourci | Effet                                                        |
| --------- | ------------------------------------------------------------ |
| `⌥ + 2`   | Divise la fenêtre courante. On peut choisir entre « vertical » ou « horizontal ».<br />Ou pour focusser sur la deuxième fenêtre quand le split est déjà actif. |
| `⌥ + r`   | « r » comme « rotate », « tourner ». Pour passer de la division verticale (deux évènemenciers côte à côte) à la division horitontale (deux évènemenciers l’un sur l’autre). |
| `⌥ + 1`   | Pour focusser sur la première fenêtre.                       |
| `⌘ + ←`   | Pour focusser sur l’autre fenêtre.                           |
| `⌥ + 0`   | Pour retourner à la fenêtre unique (plus de plit). Note : C’est la fenêtre qui a le focus qui reste. |
