## Les fenêtres

Pour une navigation encore plus pratique, on peut diviser la fenêtre principale pour obtenir deux évènemenciers affichés en même temps, l’un au-dessous de l’autre ou l’un à côté de l’autre.

Raccourcis pour opérer la division et en sortir :

| Raccourci | Effet                                                        |
| --------- | ------------------------------------------------------------ |
| `⌥ + 2`   | Divise la fenêtre courante. On peut choisir entre « vertical » ou « horizontal ».<br />Ou pour focusser sur la deuxième fenêtre quand le split est déjà actif. |
| `⌥ + r`   | « r » comme « rotate », « tourner ». Pour passer de la division verticale (deux évènemenciers côte à côte) à la division horitontale (deux évènemenciers l’un sur l’autre). |
| `⌥ + 1`   | Pour focusser sur la première fenêtre.                       |
| `⌘ + ←`   | Pour focusser sur l’autre fenêtre.                           |
| `⌥ + 0`   | Pour retourner à la fenêtre unique (plus de plit). Note : C’est la fenêtre qui a le focus qui reste. |

### Fenêtre active

C’est toujours la fenêtre active (celle qui a le focus) qui reçoit les commandes clavier. Elle se reconnait au fait que l’autre est moins opaque et n’est pas entourée d’un filet vert discret.

## Les panneaux

On trouve également un grand nombre de panneaux dans *EventerX*, panneau des [brins](04-Brins.md), panneau des [personnages](05-Persos.md), etc.

La plupart de ces panneaux peuvent être déplacés à l’aide du raccourci `⌃⇧` suivi des flèches dans les quatre direction.
