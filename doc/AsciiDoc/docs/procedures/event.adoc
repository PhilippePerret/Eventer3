=== Event ===

_(Event = évènement)_

==== Obtenir de l'aide sur la liste ====

[cols="1,4"]
|====
| **Action**      | **Attendu**
| combo:maj+?[]   
| Le panneau de l'aide contextuelle s'ouvre +
Vous pouvez prendre connaissance du raccourci à utiliser. +
(ainsi que découvrir les autres raccoucis que vous ne connaissez peut-être pas)
| combo:tab[] x 2  | Le bouton combo:Fermé[] est actif
| combo:enter[]  | L’aide contextuelle est fermé
|====

==== Jouer un raccourci depuis l'aide contextuelle ====

[cols="1,4"]
|====
| **Action**      | **Attendu**
| combo:maj+?[]   
| Le panneau de l'aide contextuelle s'ouvre +
| combo:fb[]/combo:fh[] x   | Sélectionne le raccourci voulu
| combo:tab[] | Le bouton combo:Jouer[] est actif +
(Noter qu'en fait il n’est pas nécessaire d'être sur le bouton combo:Jouer[] pour activer la raccourci, il suffit de presser comb:enter[] pour jouer le raccourci sélectionné)
| combo:enter[]  | Le raccourci est joué

|====


==== Modifier un évènement ====

[cols="1,4"]
|====
| **Action**      | **Attendu**
|                 | On doit se trouver dans l'évènemencier d'un projet.
| combo:enter[]   | L'event passe en édition (fond vert)
| combo:a-z[]     | On modifie son texte (appelé aussi “titre”)
| combo:tab[]     | Propriété suivante sélectionnée (statut)
| combo:fb[]      | Le menu des status s'ouvre
| combo:fb[]/combo:fh[] | sélectionne une valeur
| combo:enter[]   | Choisit cette valeur
| combo:tab[]     | Propriété suivante sélectionnée (météo)
| idem            | Une valeur est choisie (ou pas)
| combo:tab[]     | Propriété suivante sélectionnée (effet)
| idem            | Une valeur est choisie (ou pas)
| combo:maj+tab[] | Propriété précédente sélectionnée (météo)
| combo:tab[]x2   | Propriété suivante sélectionnée (lieu)
| idem            | Une valeur est choisie (ou pas)
| combo:enter[]   | Fin de l'édition de l'event (automatiquement enregistré)
|====

==== Choisir les brins de tous les events cochés ====

[cols="1,4"]
|====
| **Action**      | **Attendu**
|                 | Un évènemencier est ouvert +
Le projet possède plusieurs brins +
Plusieurs évènements sont cochés
| combo:maj+b[]   | La fenêtre des brins est ouverte +
_(tous les brins se sont mis en pseudo édition)_
| combo:fh/fb[]   | Sélectionne le brin voulu
| combo:space[]   | Coche ou décoche le brin selon son état précédent +
Le badge du brin s'ajoute ou se retire de l'event sélectionné (mais pas des autres)
| combo:cmd+enter | La fenêtre des brins est refermée +
Les brins se sont ajoutés et retirés à tous les events cochés.
|====



==== Choisir les personnages de l'event ====

[cols="1,4"]
|====
| **Action**      | **Attendu**
|                 | Choisir un projet qui xref:procedure-creation-personnage[contient des personnages]
| combo:p[]       | La fenêtre des personnages est ouverte
| combo:fh/fb[]   | Le personnage voulu est sélectionné
| combo:space[]   | Le personnage est coché +
Et son avatar ou son badge s'affiche sur la ligne de l'event.
| … |
| combo:p[] \| combo:cmd+enter[] | La fenêtre des personnages est fermée
|====

[cols="1,4"]
|====
| **Action**        | **Attendu**
|                   | Un évènemencier doit être affiché, avec plusieurs events.
| combo:fb+space[]  | Plusieurs events sont cochés
| combo:maj+p[]     | Ouvre le panneau des personnages en mode “pour tous les cochés”
|====

==== Définir l'aspect d'un event ====

////
à faire :

////

[cols="1,4"]
|====
| **Action**      | **Attendu**
|                 
| Sélectionner un event qui possède a) au moins deux brins, b) une météo et un effet (jour, nuit, etc.) défini.

| combo:o[]       | La panneau des options est ouvert
| combo:fb[] x    | L'option “Couleur de l'évènement” est sélectionné
| combo:tab[]     | Le menu de l'option est actif
| combo:fb[] x    | Choisir “Couleur propre de l'event”
| combo:o[]       
| Le panneau des options se ferme +
  La couleur propre à l'event est appliquée
| combo:o[]       | La panneau des options est ouvert
| combo:fb[] x    | L'option “Couleur de l'évènement” est sélectionné
| combo:tab[]     | Le menu de l'option est actif
| combo:fb[] x    | Choisir “Couleur du premier brin”
| combo:o[]       
| Le panneau des options se ferme +
  La couleur du premier brin est appliquée
| combo:o[]       | La panneau des options est ouvert
| combo:fb[] x    | L'option “Couleur de l'évènement” est sélectionné
| combo:tab[]     | Le menu de l'option est actif
| combo:fb[] x    | Choisir “Couleur de la météo”
| combo:o[]       
| Le panneau des options se ferme +
  La couleur de la météo est appliquée

|                 | La couleur de la météo de l'event est appliquée
|====
