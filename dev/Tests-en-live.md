## Depuis le lancement de l’application

### Créer complètement un nouveau projet

- « n » => Filepicker apparait

- Se rendre dans le dossier « Documents »

- « n » => champ pour créer un nouveau dossier

- « Mon nouvel eventer » + Enter

  => Nouveau projet et dossier créés

### Retrait d’un projet

- « ⌦ » sur le nouveau projet

  => il disparait

### Ouvrir un projet

- « n » => filepicker s’ouvre
- choisir le dossier qu’on vient de créer et retirer
