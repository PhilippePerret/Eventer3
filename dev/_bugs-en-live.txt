SYSTÈME FIN D'AIDE (system/help.js)
Fonctionnement : 
- Un message d'erreur est affiché (ERRORS[2000] par exemple). 
- Or, une aide est définie pour ce message (AIDE['E2000']) 
- Si l'user joue "?" (Maj+","), un pré-panneau s'ouvre : 
"Voulez-vous voir : 
(f) l'aide du message
(j) l'aide du contexte courant" 
(c'est un KeyboardablePanel).
- l'user choisit "f" => Un panneau d'aide s'ouvre avec le texte de l'aide en question. Normalement a priori, ce sera en markdown (ou, si j'ai le courage, un format plus adapté qui permettra de distinguer une action à faire, un effet, etc.).
- de la même manière, quand il y a un changement de contexte (voir où il est déjà indiqué à l'aide contextuelle), l'help.js doit être appelé pour savoir s'il existe une aide pour ce contexte. Si oui, "?" permet de l'afficher (avec choix si une autre aide, une aide-message, est aussi déclaré — voir )
Une "aide-message" disparait dès que le message disparait.

Maj+, pressé => si plusieurs aides activées (message/erreur ou contexte) => menu popup pour choisir => rejoindre l'aide appropriée
AIDE['E6100'] = {file: "Aide.md", anchor: "ancre-a-rejoindre"}
