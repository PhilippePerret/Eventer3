Toujours dire **"J'ai terminé"** quand une tâche est finie.

**Why:** L'utilisateur a explicitement demandé ce signal clair pour savoir quand agir (lancer les tests, etc.).

**How to apply:** Chaque fois qu'une implémentation, correction ou modification est complète, terminer le message par "J'ai terminé." — sans fioriture, sans "les tests peuvent être relancés", juste ce signal.
