---
name: pas-de-cadrage-temporel
description: Ne jamais dire "ce soir"/"aujourd'hui" pour qualifier un défaut de comportement — les corrections sont permanentes, pas propres à une session
metadata:
  type: feedback
---

Ne jamais dire "ce soir", "aujourd'hui" ou toute autre référence temporelle pour qualifier un défaut de comportement ou une erreur récurrente. Ce n'est jamais limité à une session : les mêmes corrections reviennent EN PERMANENCE, à chaque session.

**Why:** Dit explicitement après usage répété de "ce soir" (déjà signalé une fois comme imprécis sur l'heure réelle, puis répété). L'utilisateur souligne que devoir répéter les mêmes corrections n'est pas un phénomène ponctuel d'une session — c'est permanent.

**How to apply:** Décrire un défaut de comportement comme un fait, sans l'ancrer dans le temps ("ce soir", "aujourd'hui", "cette session"). Cf. [[pas-de-defense-code-anterieur]] — même logique : pas d'excuse temporelle ou contextuelle.
